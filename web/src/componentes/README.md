# Componentes (UI reutilizable)

| Carpeta | Qué agrupa |
|--------|------------|
| **`layout/`** | Piezas de **marco global** de la app (cabecera, barra superior). No dependen de un dominio de negocio concreto. |
| **`exposicion/`** | UI ligada al dominio **exposición**: búsqueda y anotación de ejemplares en una expo. |
