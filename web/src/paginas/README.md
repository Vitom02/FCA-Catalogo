# Páginas (pantallas con ruta)

Cada subcarpeta agrupa pantallas según **qué datos o flujo** tratan.

| Carpeta | Datos / dominio |
|--------|------------------|
| **`autenticacion/`** | Usuario y contraseña, sesión (login). |
| **`catalogo/`** | Listado y filtros del catálogo de **exposiciones** (tabla principal). |
| **`exposicion/`** | Una **exposición** concreta: anotación de ejemplares (detalle por URL). |
